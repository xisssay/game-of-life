# Игра «Жизнь»

Симулятор клеточного автомата Конвея в терминале на C.

Поле 30×30 с тороидальной топологией (края замкнуты). В центре поля задаётся случайная начальная конфигурация размером от 3×3 до 10×10. Программа умеет автоматически определять тип развития симуляции:

- **Вымирание** — все клетки погибают
- **Стабильная** — конфигурация перестаёт меняться
- **Периодическая** — конфигурация циклически повторяется
- **Долгоживущая** — активна 100+ поколений

## Запуск

```bash
git clone https://github.com/xisssay/game-of-life
cd game-of-life
make
./life
```

**Требования:** GCC, Linux/macOS

## Меню

| Пункт | Описание |
|-------|----------|
| `1` | Случайная симуляция с визуализацией |
| `2` | Автопоиск всех 4 типов конфигураций |
| `3` | Загрузить сохранённую конфигурацию |

Найденные конфигурации сохраняются в файл `life_configs.txt` и могут быть запущены повторно для демонстрации.
